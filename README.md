# “海棠”系列丛书 / Begonia_Series

在看目前论文的过程中我经常发现很多基本概念不是很清楚，而论文是不会详细介绍这些概念的，这就导致我很多时候看论文看的很糊涂。因此该系列的书籍（或者说文档）将聚焦于人工智能相关基本概念的阐述，内容多来源于我日常的笔记，希望对大家能有所帮助。

## 神经计算的数学原理 / Mathematical Principles of Neural Computing

该书籍计划详细阐述神经计算的数学原理，目的在于尽可能地给出详细而清晰的推导以及渐进式的学习曲线。书中只会涉及一些核心概念，防止增加新手读者的困惑。目前更新的目录如下：

- 1. 回归模型
  - 1.1 线性模型
  - 1.2 量化评价
  - 1.3 模型求解
  - 1.4 线性模型求解二分类问题
  - 1.5 多项式回归
  - 1.6 正则化处理
  - 1.7 算法实验
- 2. 梯度下降
  - 2.1 随机梯度下降
    - 2.1.1 求解线性二分类问题
  - 2.2 小批量随机梯度下降
- 3. 感知器和神经网络

- 1. 导论
- 2. 贝尔曼公式
